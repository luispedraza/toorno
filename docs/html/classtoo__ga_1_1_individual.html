Descripción detallada

Cada uno de los individuos del algoritmo genético Su carga genética indica una solución de asignación propuesta Está implementado como una lista de DEFAULT_PEOPLE listas, cada una de las cuales es la asignación de días y turnos para una personas.

Siendo d=persona, d=día, t=turno: chromosomes: [ p_1[t_1_1,t_1_2,…,t_1_nshits,t_2_1,t_2_2,…,t_2_nshifts,…,t_ndays_1,t_ndays_2,…,t_ndays_nshifts], p_2[t_1_1,t_1_2,…,t_1_nshits,t_2_1,t_2_2,…,t_2_nshifts,…,t_ndays_1,t_ndays_2,…,t_ndays_nshifts], … p_npeople[t_1_1,t_1_2,…,t_1_nshits,t_2_1,t_2_2,…,t_2_nshifts,…,t_ndays_1,t_ndays_2,…,t_ndays_nshifts], ]
